This repository contains a Python GUI application designed for the analysis of 
the polarization and phase images obtained with the microscope described in the following publications:

[1] H. Laviec, M. Dubreuil, S. Rivet et al., "Widefield quantitative polarized light microscopy using spectrally encoded null polarimetry", Optics Letters, 2024 \n

[2] H. Laviec, S. Rivet, M. Dubreuil et al. "Video rate widefield polarized light microscopy by spectrally structured illumination", Optics Letters, 2025
